Introducción


Este proyecto se realizó con el objetivo de mejorar los conocimientos de estructuración de datos al nivel de árboles.
Para este proyecto se solicitó que se realizara en C++ usando Visual Studio 2022 Community como único IDE, y se facilitará un diagrama de clases usando GitMind.
Se construirá un sistema hospitalario donde se pueden asignar citas para pacientes, esos pacientes obtienen una cita con un doctor en particular. Cuando se crea la cita se genera cuna factura, el doctor generará una receta medica en donde cada receta contiene el detalle de varios medicamentos y sus respectivas dosis.



Objetivo


El Principal de este proyecto es aplicar los conocimientos teóricos, principios y técnicas vistas durante la clase haciendo uso del sistema de estructuración de datos en árboles, es importante destacar durante la realización de este proyecto que el código correctamente estructurado y fácil de poder reutilizar.



Instalación


Para la ejecución de este programa se necesita un editor como Visual Studio Community 2022 o Visual Studio Code.

Para Visual Studio Community 2022 necesitas tener descargado la carga de trabajo Desarrollo para el escritorio con C++, ya con esto será suficiente para poder ejecutar el programa.

En el caso de Visual Studio Code necesitas tener instalado en tu computadora un compilador de C++ que te permita correr el programa, los más recomendados son g++ o clang.

Se recomienda tener instalado también la extensión de C++ en Visual Studio Code para la detección de errores de sintaxis y ayudas con las librerías.
Una vez instalado, pulsas en el botón de ejecutar en la esquina superior derecha.
Si es tu primer programa de C++ corriendo en Visual Studio Code se te abrirá la paleta de comando pidiendote que selecciones un compilador para debugear el programa. Te recomiendo que selecciones alguno de los 2 compiladores que están marcados.
Ya que lo tengas seleccionado, Visual Studio Code empezará a compílar el programa. El archivo principal del programa se llama main.cpp



Descripción


Cuando inicies el programa serás recibido con una pantalla de inicio que despliega todas las funciones del menú incluyendo la opción de salir del programa.
Podrás utilizar las teclas de arriba y abajo para desplazar entre las opciones, y pulsar ENTER para ingresar a la opción deseada.

  • Ingresar Doctor:

    Al seleccionar está opción se le solicitará ingresar el número de cédula del doctor a registrar, la cual deberá de ser de 9 dígitos numéricos de extensión esta opción no permite ingresar una cédula ya registrada.
    Posteriormente se te solicitará el nombre completo del doctor, seguidamente se te presentará una pantalla en al cual podrás utilizar las teclas de arriba y abajo, junto con la tecla ENTER para seleccionar la correspondiente al doctor.
    En las siguientes pantallas podrás ingresar la dirección del doctor, y las horas de entrada y salida de este, las cuales se pueden ajustar por medio de las teclas de arriba y abajo, y seleccionarlas con la tecla ENTER.
    Por último, se te solicitará ingresar el número telefónico del doctor, el cual debe constar de 8 dígitos numéricos válidos.
    En primera instancia, el sistema te mostrará una pantalla con el resumen de los datos a ingresar, ante el cual se te solicitará confirmar los datos con "Sí" o "No". De seleccionar "Si" el nuevo doctor será guardado y registrado en el sistema y en el archivo "datosDoctor.dat", de lo contrario se le solicitará ingresar nuevamente la información del doctor.


  • Ingresar Paciente:

    En caso de seleccionar la segunda opción, se te solicitará ingresar el número de cédula del paciente que deseas ingresar, la cual deberá de ser de 9 dígitos numéricos de extensión, esta opción no permite ingresar una cédula ya registrada.
    Posteriormente se te solicitará el nombre completo del paciente, su dirección y número telefónico, el cual debe constar de 8 dígitos numéricos válidos.
    Seguidamente, el sistema te mostrará una pantalla con el resumen de los datos a ingresar, ante el cual se te solicitará confirmar los datos con "Sí" o "No".
    De seleccionar "Si" el nuevo paciente será guardado y registrado en el sistema y en el archivo "datosPaciente.dat", de lo contrario se le solicitará ingresar nuevamente la información del paciente.


  • Ingresar Medicamentos:

    Al seleccionar la tercera opción, se te solicitará ingresar el código del medicamento que se desea añadir, el cual deberá constar de únicamente dígitos numéricos, esta opción no permite ingresar un código ya registrado.
    Posteriormente se te solicitará el nombre del medicamento, seguido de la cantidad disponible en inventario, esta opción permite ingresar únicamente dígitos numéricos.
    Ahora, el sistema te mostrará una pantalla con el resumen de los datos del medicamento a ingresar, ante el cual se te solicitará confirmar los datos con "Sí" o "No".
    De seleccionar "Si" el nuevo paciente será guardado y registrado en el sistema y en el archivo "datosMedicamento.dat", de lo contrario se le solicitará ingresar nuevamente la información de este.


  • Registrar Cita:

    Al seleccionar esta opcion se te solicitará ingresar el número de cédula del paciente a quien le corresponde la cita, y en la siguiente pantalla se te solicitará la cédula del doctor con quien se desea la cita, ambas deberán ser de 9 dígitos numéricos de extensión, esta opción válida que las cédulas ya estén registradas en el sistema.
    Luego se te solicitará ingresar le fecha en que deseas la cita, para esto podrás utilizar las teclas de arriba y abajo, junto con la tecla ENTER para seleccionar el día, mes y año deseados.
    Posteriormente, se te mostrará el día de la cita ingresado y los horarios disponibles para la misma, ante el cual se te solicitará confirmar los datos con "Sí" o "No".
    De seleccionar "No" se te volverá a preguntar el día de la cita, de lo contrario, se te redirigirá a una pantalla en donde debes ingresar, por medio de las teclas de arriba y abajo, la hora deseada para la cita y seleccionarla con la tecla ENTER.
    Por último, el sistema te mostrará una pantalla con el resumen de la cita, nótese que acá se muestra el código único de la cita que generó el sistema de manera automática.
    Se te solicitará confirmar los datos con "Sí" o "No". De seleccionar "Si" la cita será guardada y registrada en el sistema y en el archivo "datosCita.dat", de lo contrario se le solicitará ingresar nuevamente la información.


  • Atender Cita:

    Si seleccionas esta opción, se te solicitará ingresar código de la cita que se desea atender, el cuál es de carácter numérico y debe estar ya registrado en el sistema.
    Al hacerlo, el sistema te mostrará una pantalla con el resumen de la cita ingresada, ante el cual se te solicitará confirmar si deseas atenderla con "Sí" o "No". De seleccionar "Si", se procede a preguntar sobre el diagnóstico resultante de la cita, así como el código del medicamento a recetas, el cual ya debe de existir en el sistema.
    También, se le solicitará la cantidad (de carácter numérico) de medicamento, y la dosis en que debe de tomarse.
    Ahora, el sistema te mostrará una pantalla con el resumen de la receta y se te solicitará confirmar los datos con "Sí" o "No". De seleccionar "No" se te volverán a consultar los datos para la receta, de lo contrario, atenderá la cita y será eliminada de las citas pendientes del sistema y del archivo "datosCita.dat".
    Además, se creará un registro de la receta expuesta, la cual se guardará en el sistema y en el archivo "datosReceta.dat". De haber seleccionado "No" en el primer resumen, se le solicitará ingresar nuevamente el código de la cita a atender.


  • Modificar Cita:

    Si seleccionas esta opción, se te solicitará ingresar el código de la cita que deseas modificar, el cuál es de carácter numérico y debe estar ya registrado en el sistema.
    Al hacerlo, el sistema te mostrará una pantalla con el resumen de la cita ingresada, ante el cual se te solicitará confirmar si deseas modificarla con "Sí" o "No". De seleccionar "Si", se te solicitarán nuevamente los datos de la Opción 4 y la cita será modificada en el sistema y en el archivo "datosCita.dat", de lo contrario se le solicitará ingresar nuevamente el código de la cita a modificar.


  • Cancelar Cita:

    Al seleccionar esta opción, se te solicitará ingresar el código de la cita que deseas modificar, el cuál es de carácter numérico y debe estar ya registrado en el sistema.
    Al hacerlo, el sistema te mostrará una pantalla con el resumen de la cita ingresada, ante el cual se te solicitará confirmar si deseas cancelarla con "Sí" o "No". De seleccionar "Si", la cita será cancelada y eliminada en el sistema y del archivo "datosCita.dat", de lo contrario se le solicitará ingresar nuevamente el código de la cita a cancelar.


  • Consultar Factura:

    Cuando seleccionas esta opción, se te solicitará ingresar el código de las facturas a consultar, el cuál es de carácter numérico y debe estar ya registrado en el sistema.
    Al hacerlo, el sistema te mostrará una pantalla con el resumen de la factura expuesta.
    Posteriormente, el sistema te preguntará si deseas consultar otra receta, con "Sí" o "No". De seleccionar "Si", te volverá a consultar el código de otra factura, de lo contrario, el sistema regresará al menú principal.


  • Consultar Receta Médica:

    Cuando seleccionas esta opción, se te solicitará ingresar el código de la receta a consultar, el cuál es de carácter numérico y debe estar ya registrado en el sistema.
    Al hacerlo, el sistema te mostrará una pantalla con el resumen de la receta expuesta. Posteriormente, el sistema te preguntará si deseas consultar otra receta, con "Sí" o "No". De seleccionar "Si", te volverá a consultar el código de la otra receta, de lo contrario, el sistema regresará al menú principal.


  • Mostrar Arbol:

    En caso de seleccionar esta opción, el sistema te mostrará una pantalla donde se te solicita seleccionar el árbol que desees ver: doctores, pacientes, medicamentos, citas o receta médica.
    Podrás utilizar las teclas de arriba y abajo para desplazarte entre las opciones y seleccionar la que deseas con la tecla ENTER.
    Seguidamente el sistema mostrará los códigos de los registros ordenados de dicha estructura de árbol por post-orden.


  • Recorrido Profundo:

    Para esta opción, el sistema te mostrará una pantalla donde se te solicita seleccionar el árbol que desees ver: doctores, pacientes, medicamentos, citas o receta médica.
    Podrás utilizar las teclas de arriba y abajo para desplazarte entre las opciones y seleccionar la que deseas con la tecla ENTER.
    Seguidamente el sistema mostrará los códigos de los registros ordenados de dicha estructura de árbol por pre-orden.


  • Recorrido por Niveles:

    Al seleccionar esta opción, el sistema te mostrará una pantalla donde se te solicita seleccionar el árbol que desees ver: doctores, pacientes, medicamentos, citas o receta médica.
    Podrás utilizar las teclas de arriba y abajo para desplazarte entre las opciones y seleccionar la que deseas con la tecla ENTER.


  • Altura del Arbol:

    En caso de seleccionar esta opción, el sistema te mostrará una pantalla donde se te solicita seleccionar el árbol al que le deseas conocer la altura: doctores, pacientes, medicamentos, citas o receta médica.
    Podrás utilizar las teclas de arriba y abajo para desplazarte entre las opciones y seleccionar la que deseas con la tecla ENTER.
    Seguidamente el sistema mostrará el valor de la altura de dicho árbol.


  • Contar Nodos:

    Al seleccionar esta opción, el sistema te mostrará una pantalla donde se te solicita seleccionar el árbol al que le deseas conocer la cantidad de nodos: doctores, pacientes, medicamentos, citas o receta médica.
    Podrás utilizar las teclas de arriba y abajo para desplazarte entre las opciones y seleccionar la que deseas con la tecla ENTER.
    Seguidamente el sistema mostrará la cantidad de nodos presentes de dicho árbol.


  • Contar Hojas:

    Al seleccionar esta opción, el sistema te mostrará una pantalla donde se te solicita seleccionar el árbol al que le deseas conocer la cantidad de hojas: doctores, pacientes, medicamentos, citas o receta médica.

    Seguidamente el sistema mostrará la cantidad de hojas presentes de dicho árbol.


  • Salir:

    Al seleccionar esta opción, el sistema te mostrará una pantalla donde se te solicita seleccionar el árbol al que le deseas conocer la cantidad de hojas: doctores, pacientes, medicamentos, citas o receta médica.
    Podrás utilizar las teclas de arriba y abajo para desplazarte entre las opciones y seleccionar la que deseas con la tecla ENTER.
    Seguidamente el sistema mostrará la cantidad de hojas presentes de dicho árbol.



Curiosidades


En el codigo fuente encontrarás muchas refencias en los cout con respecto a un comando como este \033[100m, este comando es el encargado de cambiar el color a la consola apartir de donde va a escribir, o imprimir imformación el cout, entonces para saber la paleta de colores acontinuacion hay una imagen con la paleta de colores.
Nota: Foreground se refiere al color de la letra, y Background se refiere al color del fondo.

Tambien durante el codigo, en el menú, para cuando se selecciona una opcion, se le asignar un color, pero ese color viene en formato decimal, para eso, hay otra tabla de colores si deseas cambiar el color de como se muestra la opcion marcada.
Este viene en formato Hexadecimal, por lo que hay que convertir el valor que desees en decimal. En la calculadora de Windows, puedes convertir ese valor seleccionando el modo programador de la calculadora.
Y si seleccionas donde dice HEX, ahí podrás ingresar ese digito y se convertirá de forma automatica a decimal, el primer digito que ingreses va a ser para el fondo y el segundo va a ser para la letra.
